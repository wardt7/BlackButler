AA = .-.- = New Line (space down one line)
AR = .-.-. = New Page
BT = -...- = New Paragraph (space down two lines)
DO = -..--- = Change to Wabun (japanese) code
SK = ...-.- = End of contact
SN = ...-. = Change to morse code (from wabun)
